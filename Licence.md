À propos de ce projet
Ce projet de base de données a été réalisé dans le cadre du cours 420-3N3-DM Bases de données au Cégep de Drummondville.
Auteure : Leticia Monsoh
Cours : Techniques de l'informatique
Institution : Cégep de Drummondville
Date : Décembre 2025
Objectif académique
Cette base de données pour le Gym Proactif Drummondville a été conçue à des fins éducatives pour démontrer :

La modélisation de bases de données relationnelles
L'implémentation avec PostgreSQL
Les bonnes pratiques en SQL
La documentation de projets informatiques

Utilisation
Ce projet est disponible sous licence MIT pour permettre à d'autres étudiants et développeurs de :

Étudier la structure de la base de données
S'inspirer pour leurs propres projets
Contribuer à l'amélioration du modèle

Avertissement
Les données utilisées dans ce projet sont fictives. Toute ressemblance avec des personnes réelles serait purement fortuite. Les tarifs mentionnés sont basés sur les informations publiques du Gym Proactif au moment de la création du projet.
